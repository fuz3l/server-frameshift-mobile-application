```bash
pip install -r requirements.txt
```

### 2. Set Up Environment

```bash

# Edit .env with your configuration
```

### 3. Initialize Database

```bash

flask db upgrade
```

### 4. Run the Application

```bash
python app.py
```

The application will be available at `http://localhost:5000`
